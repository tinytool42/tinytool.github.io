About TinyTool

TinyTool is a simple Windows batch utility that helps you clean temporary files, list system info, manage startup apps, and run a speed test — all via a minimal console interface.

Features

  • Clear temporary files quickly
  • Display IPv4/IPv6, MAC address, Windows version, and proxy settings
  • List and disable startup applications
  • Direct link to speedtest.net for internet speed testing
