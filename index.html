About TinyTool

TinyTool is a lightweight Windows batch utility that lets you clear temporary files, view system info, manage startup apps, and run speed tests from your console — all with ease.

Designed by mariocrazyy, TinyTool aims to simplify your Windows maintenance tasks.

Features

  • Clear temporary files quickly
  • Show IPv4, IPv6, MAC address, Windows version, and proxy status
  • List and disable startup apps
  • Launch speedtest.net directly
